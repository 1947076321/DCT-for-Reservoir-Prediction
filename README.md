What is this repository for?
DCT-for-Reservoir-Prediction is a reservoir prediction framework based on semi supervised co-training.
Usage
Divide your own data into two parts according to the training and testing sets, and refer to the. npz file for the specific format.
Run the train.py program,When the program runs, it will print the matching rate of the test set, and when the program terminates, it will save the .pth model weight file
